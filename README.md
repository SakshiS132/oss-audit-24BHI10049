Project Description
This project is an audit of an open-source software system. It explores its origin, license, Linux usage, ecosystem, and compares it with proprietary alternatives. It also includes 5 shell scripts demonstrating Linux and automation skills.

Scripts Overview

1. script1.sh – System Identity Report
Displays system information like kernel version, user, uptime, and OS details.

2. script2.sh – FOSS Package Inspector
Checks if selected software is installed and shows version and details.

3. script3.sh – Disk and Permission Auditor
Analyzes directories for size, permissions, and ownership.

4. script4.sh – Log File Analyzer
Reads log files and counts occurrences of keywords like ERROR.

5. script5.sh – Open Source Manifesto Generator
Generates a personalized open-source philosophy file based on user input.
